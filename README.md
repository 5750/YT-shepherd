# YT-shepherd
YT Shepherd is a Firefox add-on that lets you categorize YouTube channels as Recommended, Neutral, or Discouraged. It visually modifies thumbnails to reflect your preferences â€” guiding your attention and helping you stay productive, one tile at a time.

## Installation
Download the repository: Code -> Download ZIP
Then, in Firefox, go to about:debugging, click "This Firefox", then "Load Temporary Add-on" and select the extension file.
